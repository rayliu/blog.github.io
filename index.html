Hello CoffeeScript

Hello CoffeeScript is a simple CoffeeScript tutorial comprised of self-explanatory "hello world" examples of increasing complexity. It was designed to provide a smoother transition from zero to the popular Todos example.

CoffeeScript is a programming language that transcompiles to JavaScript. The language adds syntactic sugar inspired by Ruby, Python and Haskell to enhance JavaScript's brevity and readability, and add more sophisticated features like list comprehension and pattern matching.

The tutorial starts with a minimalist online view sample, and progressively introduces Functions, Classes, Inheritance, and Super.

Start the tutorial

Once in the tutorial, use the navigation menu in the top-right corner to view other examples. Example numbers are in order of increasing complexity.
